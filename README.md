# Python Practice Programs

Three simple Python programs demonstrating basic programming concepts.

## Programs Included

### 1. Simple Calculator (`calculator.py`)
Basic arithmetic calculator that takes two numbers and an operator (+, -, *, /) as input.
- Handles division by zero
- Validates operator input
- Supports decimal numbers

### 2. Random Number Generator (`random_number.py`)
Generates and displays a random number between 1 and 100.
- Uses Python's `random` library
- Includes example of generating multiple random numbers

### 3. Square Root Calculator (`square_root.py`)
Calculates the square root of a given number.
- Uses `math.sqrt()` function
- Handles negative numbers (prevents imaginary results)
- Displays results with decimal formatting

## How to Run

```bash
# Run any program using Python 3
python calculator.py
python random_number.py
python square_root.py
